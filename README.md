## ⚙️ Características

- Launcher portable y ligero (sin instalación)

- Sistema de iconos optimizado y automático

- Tema claro y oscuro
- Interfaz rápida y fluida
- 🌍 Multiidioma (Español / Inglés)
- Modo demo con limitaciones
